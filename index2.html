Project Name: WorldSurf

Project Description

Using usability tests is something I have never experienced personally, but it seems like a very straight forward method to get direct feedback from actual users. We need to more people to know about this so that others can use this to help them get the usability help that they need. Although it also seems like usability should be a straight forward concept, while I was doing some of the usability tests to get my karma points I realized that it's not always clear what the pages are for or even clear how to login. I will definitely keep this website in mind as it will surely come in handy in the future.
